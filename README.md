# CoReviewer —— <sub>代码 Review，不该是一场对话。</sub>



## 诞生背景：

在**高强度**使用 **AI Coding** 的过程中，为了保证 coding 方向不走偏、能够正确指挥 AI 完成项目的下一步动作，我们需要理解：

- 项目代码的**整体架构**
- 代码的**运行流程**
- 局部**模块功能**
- 少量的**细节语法**

单纯使用** Claude Code 等 chat 工具**虽然可以获得十分有效的回答，几乎可以通过不断的对话解决所有疑惑，但 chat 交互方式存在天然缺陷：

- 交流效率不够高——碎片化问答导致上下文浪费
- 用户阅读体验不够好——对话流中信息与代码混杂，视觉混乱
- 缺乏结构化的审查流程——回答散落在对话中，难以回溯和管理

**CoReviewer** 正是为此而生——帮助你更好地应对 **AI Coding 时代代码量爆炸、而原始 Review 交互方式令你感到不满**的场景。
